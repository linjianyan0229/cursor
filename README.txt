# Cursor重置工具使用说明

此工具用于重置Cursor编辑器的设备ID，帮助您恢复免费试用期。

## 使用方法

1. 完全关闭Cursor应用
2. 以管理员身份运行cursor-reset.exe
3. 按照提示操作，确认重置
4. 重置完成后，重新启动Cursor
5. 使用您的账号登录（可以使用原账号或新账号）

## 注意事项

- 必须使用管理员权限运行，否则无法修改注册表
- 脚本会自动备份原配置文件
- 此工具支持Windows系统

## 常见问题

Q: 提示"Too many free trial accounts used on this machine"怎么办？
A: 这正是本工具设计用来解决的问题，运行后会重置设备ID。

Q: 重置后是否会影响Cursor中的设置和插件？
A: 不会，只会重置设备识别相关的ID，不影响其他设置。

Q: 是否需要卸载Cursor？
A: 不需要，只需关闭Cursor应用即可。
